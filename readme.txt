git下载安装

使用git进行版本控制

使用git下载github项目
1.新建文件夹，文件夹内右键git bash here
2.设置用户名
$ git config --global user.name "runoob"  
3.设置邮箱
$ git config --global user.email test@runoob.com
4.从github地址克隆项目
$ git clone <版本库的网址>
例如： $ git clone https://github.com/Inotbird/zhiliao.git

提交本地项目到github
